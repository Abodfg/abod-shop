📝 خطوات الرفع على GitHub Pages:

  1. اذهب إلى github.com/abodfg/abod-shop
  2. احذف الملف القديم index.html من المستودع
  3. اضغط Add file → Upload files
  4. اسحب الملف المحمل (سيتم تسميته تلقائياً index.html)
  5. اكتب رسالة Commit: "تحديث المتجر - ميزات جديدة"
  6. اضغط Commit changes
  7. انتظر دقيقة واحدة
  8. افتح abodfg.github.io/abod-shop
  9. اضغط Ctrl+Shift+R للتحديث
